# Rock-Paper-Scissors-Game
This repository contains a Java program that implements the classic game of Rock Paper Scissors, allowing users to play against the computer. The game is designed to be simple yet entertaining, providing an interactive experience for players of all ages.

**Features**
* User-friendly command-line interface: The game is played through the terminal, making it accessible and easy to play on any system.
* Randomized computer moves: The computer's moves are generated using the java.util.Random package, ensuring an unpredictable and challenging opponent.

**Getting Started**
To run the game on your local machine, make sure you have Java Development Kit (JDK) installed. Follow these steps:

* Clone this repository to your local machine or download the source code files.
* Open your preferred command-line interface (CLI) and navigate to the project directory.
* Compile the Java source file using the command: javac RockPaperScissors.java
* Run the program with the command: java RockPaperScissors

**How to Play**
Once the program is running, follow the on-screen instructions to play the game:

* Enter your move: Choose either rock, paper, or scissors by typing the corresponding number or keyword.
* The computer will randomly select its move.
* The game will determine the winner based on the standard Rock Paper Scissors rules.
* The result of each round will be displayed
